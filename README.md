## About

Agent Engineer focused on designing and building AI agents that solve real problems. I specialize in **Context Engineering** — crafting the right context windows, prompt architectures, and agent workflows to maximize AI capabilities.

- **Agent Design**: Building autonomous agents with clear goals and robust tool use
- **Context Engineering**: Optimizing prompts, RAG pipelines, and memory systems
- **Full-Stack AI**: From React frontends to AI backends, end-to-end delivery
